# ACE Protocol

The ACE Protocol defines the canonical, implementation‑neutral data specification for ACE‑CERT — the verifiable certificate format used within the Adaptive Carbon Engine ecosystem.

This repository contains:

- **ACE‑CERT Schema** (JSON)
- **Field definitions and SI units**
- **Validation and integrity requirements**
- **Hash‑binding rules** (input, scenario, method, calculation)
- **Versioning of the protocol**

The protocol is public, deterministic and implementation‑agnostic.  
It contains no code, algorithms or engine logic.

ACE‑CERT certificates are only valid when their hash chain is anchored in the ACE Ledger, maintained by ACE Technology.
